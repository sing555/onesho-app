# おねしょバイバイ！ぴかぴかカレンダー 開発計画表

4歳の男の子が楽しく「おねしょ卒業」を目指せる、スマートフォン向けのウェブアプリ（PWA）を開発します。

## 1. コンセプト
- **主役は子供**: 自分でスタンプを押せる、簡単で楽しい操作感。
- **褒める仕組み**: 成功した日はキラキラのスタンプ、失敗しても「次がんばろう」の優しい演出。
- **継続の工夫**: スタンプがたまるとメダルやトロフィーが出る演出。

## 2. 主要機能
- **スタンプカレンダー**: 日付をタップして、その日の状態（成功/失敗）を記録。
- **ごほうびコレクション**: 成功回数に応じてデジタルステッカーを獲得。
- **PWA対応**: スマホのホーム画面にアイコンを追加して、アプリのように使用可能。
- **データ保存**: ブラウザの LocalStorage を使用し、サーバー不要でデータを保持。

## 3. 技術スタック
- **Frontend**: HTML5, Vanilla CSS (最新のモダンな配色), JavaScript
- **PWA**: Web App Manifest, Service Worker (オフライン動作)
- **Animation**: CSS Animations, Canvas Confetti (紙吹雪演出)

## 4. 開発ステップ
1.  プロジェクト構造の作成
2.  デザイン・CSSの構築（パステルカラーと丸みのあるデザイン）
3.  カレンダー機能と記録ロジックの実装
4.  ごほうび・演出の実装
5.  PWA設定とGitHub用準備
